# Deep Learning Specialization — Coursework

Assignments and lab notebooks from the [Deep Learning Specialization](https://www.coursera.org/specializations/deep-learning) by Andrew Ng (Coursera / deeplearning.ai).

Implementations from scratch in Python and PyTorch — covering core deep learning concepts from fully connected networks to sequence models.
